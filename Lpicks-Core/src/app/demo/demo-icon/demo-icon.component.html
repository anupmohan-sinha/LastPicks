Icon Component

  • Configuration Options

    • Use this Selector in your component to include Icon Component.

              <kdi-icon [icon]="iconName">
          </kdi-icon>
  • How to use ?

    • Follow the following format to use it.

               iconName="name of the icon you want to display"; 
          
          
         Here we have used :
           iconName1="box" ;
           iconName2="search";
           iconName3="bookmark"
           iconName4="angle-left"

    The list of the icons are present in our assets/img/icons folder. Just assign the icon name to the "iconName" variable. (.svg extension after the icon name is not required)

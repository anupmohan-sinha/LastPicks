Input Field Component

Test Inputs (JSON DATA)

  • Description

    • Input Fields allow users to enter inputs into a UI. They typically appear in forms and dialogs..

  • Configuration Options

    • Use this Selector in your component to include Badge Component.

                <kdi-input-field [inputField]="demoInputField">
            </kdi-input-field>
  • How to use ?

    • Follow the following format to use it.

                 demoInputField= {  type:"name of the type of input (from the list of 'type's provided by KDI)",  
              label:"name of the label you want to display "   } ;                         
           Here we have used :
             demoInputField1= {  type: 'text',  label:'text' } ;
             demoInputField2= { type:'number', label:'number'  } ;
             and so on.

    For the list of "type"s available refer to KDI design system.

  • Configuration Options

    • Use this Selector in your component to include Card Component.

        <kdi-card>
    </kdi-card>
  • How to use?
    • The most basic card needs only an <kdi-card> element with some content. However, Angular Material provides a number of preset sections that you can use inside of an <kdi-card>
    Element Description
    <kdi-card-title> Card title
    <kdi-card-content> Primary card content. Intended for blocks of text
    <kdi-card-footer> Card Footer
